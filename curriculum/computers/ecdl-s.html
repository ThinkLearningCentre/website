ECDL Standard

Το ECDL Standard είναι μία πιστοποίηση 7-ενοτήτων που αποτελείται από τις τέσσερις πρόσφατα σχεδιασμένες βασικές ενότητες του ECDL Βάσης (η οποία πιστοποιεί τον ψηφιακό γραμματισμό ατόμου), συν τρεις επιπλέον ενότητες για την ανάπτυξη και την πιστοποίηση της ψηφιακής ικανότητας ενός ατόμου.

Σύμφωνα με τα καθιερωμένα παγκόσμια πρότυπα, η πιστοποίηση αυτή διατηρεί την δομή των 7-ενωτήτων (modules), αλλά επιτρέπει στους υποψηφίους να επιλέξουν ενότητες που ανταποκρίνονται στις ανάγκες τους.

ECDL Standard

  • Basic Modules (4)
    • Computer Essentials
    • Online Essentials
    • Word Processing
    • Spreadsheets
  • Standard Modules 3 from the following:
    • Presentation
    • Using DataBases
    • Web Editing
    • Image Editing
    • Project Planning
    • IT Security
    • Online Collaboration
    • 2D CAD
    • Health Information System usage

Computer Essentials (Απαραίτητα Υπολογιστών) - βασικές γνώσεις και δεξιότητες που σχετίζονται με τη χρήση των ηλεκτρονικών υπολογιστών και συσκευών. Αυτή η ενότητα αντικαθιστά το ECDL Core Ενότητες και 1 και 2 και περιλαμβάνει νέο περιεχόμενο.

Online Essentials - γνώσεις και δεξιότητες που διέπουν τη χρήση του Διαδικτύου για τον εντοπισμό και την ανάκτηση πληροφοριών, να συμμετάσχουν σε online δραστηριότητες, και να επικοινωνούν αποτελεσματικά. Αυτή η ενότητα αντικαθιστά το ECDL Core Module 7 και περιλαμβάνει νέο περιεχόμενο.

Word Processing (Επεξεργασία Κειμένου) - δεξιότητες που σχετίζονται με την εργασία με κείμενο σε μια εφαρμογή επεξεργασίας κειμένου. Δεν υπάρχει καμία αλλαγή στο περιεχόμενο / εξεταστέα ύλη του μαθήματος.

Spreadsheets (Υπολογιστικά Φύλλα) - δεξιότητες που σχετίζονται με την εργασία με αριθμούς σε μια εφαρμογή λογιστικών φύλλων. Δεν υπάρχει καμία αλλαγή στο περιεχόμενο / εξεταστέα ύλη του μαθήματος.
